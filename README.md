## Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/umutxyp/BlumFarmer.git
    cd BlumFarmer
    ```

2. Install dependencies:

    ```bash
    npm install
    ```

3. Build and package the app:

    ```bash
    npm run package
    ```

4. To start the app in development mode:

    ```bash
    npm start
    ```
